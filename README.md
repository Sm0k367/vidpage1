## 🌈 Welcome to the Future Club

**Drop in music, images, videos, or text and this digital club lights up in real time.
Every effect is pure code. Every session is a one-of-a-kind neon rave.**

## 🚨 Features

| Type | What You Get |
|-------------|-----------------------------------------------------------------|
| **Audio** | Neon spectrum analyzer + waveform, pulsing avatars, club grid |
| **Video** | Projector wall, music-reactive club lights and glowing bars |
| **Image** | Animated projection mapping, color-sampled lighting & ripples |
| **Text** | Kinetic UGC wall, glitch/scanline animation, rainbow typography |
| **Any file**| No backend; instant in-browser magic, drops, and resets |
| **Branding**| Custom logo everywhere, club-style favicon |
