## 🚀 What’s New?

- **Drag & drop, or upload anything:** audio, image, gif, video, text—no backend, in-browser!
- Detects file type & runs custom code-only visualizer for each (SVG/CSS/React).
- Neon dancefloor, dynamic avatars, glowing club elements—all powered by your uploaded media.
- Responsive UI, mobile-ready, deploys in 1 click to Vercel.
