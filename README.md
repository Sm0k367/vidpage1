# 🪩 **AI Lounge After Dark**
*Drop. Any. File. See the club react. Mind blown.*

---

## 🌌 What Is This?

A browser-based **audio/video/image/text visualizer nightclub**—every effect and animation is custom code.
No boring static assets. No templates. Just pure neon-powered spectacle and interactive feels:

- **Audio:** Reactive spectrum/waveform, avatar grid, lights pulse to your beats.
- **Image/GIF:** Club projection wall with color-animated environment.
- **Video:** Cyberpunk “club screen” with glowing, music-reactive lightbars.
- **Text:** Scrolling UGC wall, every character kinetic, color-shifting, glitched.

🔥 Drag & drop. Click & upload. Everything happens INSTANTLY in your browser.

---

## ✨ Features

- 🎚️ True Web Audio spectrum analyzer (with real neon, avatars, floor)
- 🎬 Video with “club light” overlays, glow synced to audio
- 🖼️ Images become projection-mapped club art, animated ripples/fog
- 📝 Text goes full UGC: Kinetic fonts, scanlines, rainbow gradients
- 🎛️ **NO** external media or asset dependencies—just React, Canvas, SVG, magic
